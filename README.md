# Developer Report

## Purpose of the Script
The purpose of this script was to create a simple task tracker that allows users to add, view, and remove tasks. This program could help users organize daily tasks or small project activities.

## Tools or Libraries Used
I used standard Python functions such as lists, loops, conditionals, and input statements. No external libraries were needed because the project was designed to remain simple and beginner friendly.

## Challenges During Development
One challenge was making sure the program handled invalid user input correctly, especially when removing tasks. I also needed to make sure the menu loop continued running properly until the user exited the program.

## Future Improvements
In future versions, I would add features such as due dates, task priorities, and the ability to save tasks to a file. I would also consider adding a graphical user interface to improve usability.

## Lessons Learned
This project helped me better understand loops, lists, and user interaction in Python. It also showed the importance of testing code and organizing program flow carefully.
